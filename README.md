# EXP-3-PROMPT-ENGINEERING-
## Aim:
Evaluation of 2024 Prompting Tools Across Diverse AI Platforms: ChatGPT, Claude, Bard, Cohere Command, and Meta

## Experiment:
Within a specific use case (e.g., summarizing text, answering technical questions), compare the performance, user experience, and response quality of prompting tools across these different AI platforms.

## Algorithm:
PHASE 1 — SETUP
Step 1: Define objective └─ Select use case (summarise / Q&A / creative)

Step 2: Select platforms └─ ChatGPT, Claude, Bard, Cohere Command, Meta LLaMA

PHASE 2 — DESIGN
Step 3: Design prompt library └─ 30 prompts × 3 use cases = 90 prompts total └─ Ensure identical wording across all platforms

Step 4: Define scoring rubric └─ 5 dimensions × 10-point scale (see Scoring Rubric)

PHASE 3 — EXECUTE
Step 5: Configure environment └─ API access for each platform └─ temperature = 0.7, max_tokens = 1024

Step 6: Run prompts
Write a PDF evaluation report comparing AI platforms — ChatGPT, Claude, Bard, Cohere Command, and Meta — on their prompting capabilities in 2024. Use three experimental use cases: text summarisation, technical Q&A, and creative content generation. Score each platform on a 10-point scale across 5 dimensions and include tables, platform profiles, cost analysis, and strategic recommendations.

PHASE 4 — EVALUATE
Step 7: Score responses └─ 5 independent reviewers apply rubric └─ Average scores per dimension per platform

Step 8: Check consistency └─ Compute variance across 5 runs per prompt └─ Flag high-variance responses for review

Step 9: Aggregate results └─ Calculate dimension means └─ Calculate overall mean per platform └─ Rank platforms per use case

PHASE 5 — OUTPUT
Step 10: Generate report └─ Comparative tables └─ Platform profiles └─ Recommendations by use case
